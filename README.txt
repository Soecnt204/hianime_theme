HiClone demo theme
This package is a cleaned, legal layout inspired by HiAnime's structure (for learning/design only).
Files:
 - index.html (homepage)
 - anime_detail.html (detail page placed in root or subfolder)
 - watch/watch.html (watch page placeholder)
 - css/styles.min.css
 - js/app.min.js, js/watch.min.js
 - images/* (placeholder PNGs)
 - watch/manifest.json (PWA manifest placeholder)

Replace images and text with your own legal content.
